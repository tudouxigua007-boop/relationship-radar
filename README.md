# 关系雷达 Relationship Radar

帮助年轻用户识别流行文化和日常对话中被浪漫化、但可能不健康的关系行为，并练习安全、清晰地表达边界。

## 一天黑客松 MVP

核心流程：

1. 输入或粘贴一段对话
2. 标出值得注意的原句
3. 解释可能的行为模式与真实含义
4. 给出温和、明确、退出三类回应
5. 进行一轮模拟式应对练习

暂不开发：登录、数据库、社区、自动读取聊天软件、模型训练和复杂语音情绪识别。

## 三人分工

- 前端：页面、结果卡片、交互和演示效果
- AI/API：模型调用、结构化输出、异常兜底
- 内容/集成：案例库、提示词测试、路演和主分支集成

建议分支：

```text
feature/frontend
feature/ai-api
feature/content-demo
```

## 建议接口

`POST /api/analyze`

请求：

```json
{
  "text": "我每天问你一遍，直到你答应。"
}
```

返回：

```json
{
  "summary": "这句话把持续施压包装成坚持。",
  "riskLevel": "warning",
  "signals": [
    {
      "quote": "直到你答应",
      "pattern": "无视拒绝",
      "meaning": "没有把拒绝视为需要尊重的决定。"
    }
  ],
  "responses": [
    {
      "tone": "明确",
      "text": "我已经表达过不愿意，请尊重我的决定。"
    }
  ]
}
```

## 安全原则

- 分析具体行为，不给人物下诊断
- 上下文不足时明确说明不确定性
- 不直接替用户决定是否分手
- 涉及威胁、暴力或跟踪时，优先提示现实安全
- 不提交真实聊天记录、`.env` 或 API Key
