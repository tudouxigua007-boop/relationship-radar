# 关系雷达 — PRD

## 产品定位

面向 18-25 岁年轻女性的关系健康检测工具。用户上传聊天截图或粘贴文本，AI 识别其中被浪漫化但实际不健康的行为模式，给出分析报告和应对建议，并支持多轮持续对话解读。

## 用户画像

- 年龄：18-25 岁女性
- 场景：刷到某段对话/台词觉得"怪怪的"，想确认自己的感觉是不是对的
- 痛点：被偶像剧/短视频浪漫化的行为模式影响判断，不确定对方的话术是"爱"还是"控制"

## 核心链路

```
输入（截图 or 文字）→ 分析报告（同页展开）→ 多轮对话（新页面）
```

## 页面结构

### 页面一：首页 + 分析报告（`/`）

**输入区域（页面上半部分）**
- Tab 切换：📷 上传截图 | ✏️ 粘贴文字
- 截图 tab：点击上传/拍照，展示预览
- 文字 tab：多行文本输入框
- 「开始扫描」按钮
- 底部：预设案例（"试试看"），用于演示和降低使用门槛

**分析报告（扫描后在页面下方展开）**
- 健康度评分：0-100 分 + 风险等级（高/中/低）+ 进度条
- 信号标注：逐句卡片，每张包含原文引用、行为模式标签（彩色）、通俗解释
- 行为模式画像：柱状进度条展示各模式占比（如控制型 80%、焦虑型 50%）
- 建议回应：温和 / 明确 / 离场三档，以卡片形式展示
- 底部 CTA：「ta 不会就此罢休？继续聊，我帮你接招 →」进入对话页

**导航**
- 顶部：logo + 右上角历史记录小图标（🕐）
- 历史记录：点击弹出轻量抽屉/弹层，展示历史 session 列表

### 页面二：对话页（`/chat`）

**入口**
- 从分析报告底部 CTA 进入（带分析上下文）
- 从历史记录恢复进入

**页面结构**
- 顶部：← 返回 + 标题
- 上下文卡片：简要展示来自分析报告的关键信息
- 对话消息列表：
  - 用户消息（右侧气泡）
  - AI 回复（左侧），包含：套路识别标签、分析说明、建议回应、鼓励反馈
- 底部输入区：
  - 预设 chip（「💡 帮我模拟 ta 可能的回复」），在输入框上方
  - 输入框 placeholder：「粘贴 ta 的回复...」
  - 发送按钮

**核心交互**
- 用户粘贴对方的新回复 → AI 分析套路 + 给应对建议
- 用户点击预设 chip → AI 模拟对方可能的回复 + 给应对建议
- 无限循环，每轮都带完整上下文

## 数据持久化

使用 localStorage 存储 session：
- 每次扫描创建一个 session（含 id、时间、输入、分析结果、对话历史）
- 杀掉应用重开不丢数据
- 历史记录从 localStorage 读取

## 安全原则

- 分析行为，不诊断个人
- 语境不足时承认不确定性
- 不替用户决定是否结束关系
- 涉及暴力/跟踪/威胁时优先提示现实安全资源

## 不做

- 用户登录/注册
- 数据库存储
- 社区功能
- 自动读取聊天应用
- 语音/情绪识别
