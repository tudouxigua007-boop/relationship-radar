# 关系雷达 — 项目讨论记录

## 项目概况

**黑客松**：一天制黑客松
**团队**：3人，都是新手，各自用 Claude/Codex vibe coding
**产品定位**：为年轻女孩做一款"关系雷达"，帮她们识别流行文化中被浪漫化但实际不健康的行为模式

## 核心功能（初步方向，待确认）

### 功能一：聊天体检
用户粘贴一段聊天记录或影视台词，AI 分析其中的不健康信号，逐句标注属于什么模式（如情感勒索、煤气灯效应、边界侵犯等），给出通俗解释和回应建议。

### 功能二：实战模拟
AI 扮演一个使用常见话术的角色（如纠缠追求型、情感绑架型、煤气灯型、控制型），用户练习识别和回应。同时有一个"教练"AI 实时点评用户的回应质量。

## 技术方案（初步建议，待确认）

- **框架**：Next.js（前后端合一，降低新手联调成本）
- **AI API**：DeepSeek（通过 OpenAI SDK 兼容接口调用）
- **样式**：Tailwind CSS
- **目标平台**：移动端优先的网页

## 分工建议

- **A（前端）**：所有页面 UI —— 首页、聊天分析页、模拟对话页
- **B（后端 + AI 集成）**：API 路由、DeepSeek 调用、prompt 和后端逻辑串联
- **C（内容 + Prompt + 演示）**：行为模式库整理、两套核心 prompt 编写、demo 案例准备、演示 PPT

## 协作方式

- 各自本地用 AI 辅助编程
- Git 直推 main，按目录分工避免冲突
- 核心约定：先定好 API 接口契约（请求/返回 JSON 格式），再各自开发
